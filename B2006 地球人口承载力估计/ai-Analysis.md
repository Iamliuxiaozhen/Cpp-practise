# 1. 翻译成数学模型  
设：  
- 𝑆：现在一开始地球上的“存量资源”（单位随便定，反正都是相对量）  
- 𝑅：地球每年“新生资源”的数量（恒定）  
- 𝑁：人口数（以“亿人”为单位）  
- 𝑐：每个人每年消耗资源的量  

那对于 
𝑁 亿人生活 𝑇 年：

总消耗 = 每年消耗𝑁⋅𝑐，共𝑇年

所以总消耗 =𝑁⋅𝑐⋅𝑇

可用资源 = 初始存量 
𝑆 + 这𝑇 年新生的资源 𝑅⋅𝑇

题目说“现有资源 + 新生资源恰好够用”，就是：
𝑆+𝑅⋅𝑇=𝑁⋅𝑐⋅𝑇
